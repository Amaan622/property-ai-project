# 🏡 Property AI - Smart Property Recommendation System

An AI-powered full-stack web application that recommends real estate properties based on user preferences using machine learning and intelligent ranking logic.

---

## 🚀 Live Demo

- 🌐 Frontend (Vercel):  
  https://property-ai-project.vercel.app

- 🔗 Backend API (Render):  
  https://property-ai-project.onrender.com/property/recommend

---

## 🧠 Features

- 🔍 AI-based property recommendation system  
- 📊 Smart ranking using score-based filtering  
- 🏠 Filters based on location, budget, BHK, sqft  
- 💬 Chat-style user interface  
- 📱 Fully responsive UI (mobile + desktop)  
- ⚡ Fast REST API backend  
- ☁️ Fully deployed on cloud (Vercel + Render)
